# 📌 Overview

Neosis Study Co-Pilot is a modern study productivity application built to improve focus, planning, and learning efficiency.

The platform combines:

* 📅 Smart study planning
* ⏱️ Pomodoro productivity tracking
* 📊 Progress analytics
* ✅ Task & schedule management
* 🎯 Goal tracking
* 🧠 AI-powered study assistance

It is designed for students, self-learners, and developers who want an all-in-one study companion.
